# 第 8 章　何时可以退休

## 核心观点

退休不是一个纯财务问题。
财务上可以用 **4% 法则** 或 **交叉点法则** 估算出你"可以退休的时间"；
但能否退休得好，取决于你**想用时间做什么**——退休后的身份认同、社交和目标，比账户余额更重要。

> 诺奖得主威廉·夏普称退休是"金融领域最棘手、最难的问题"——没有水晶球。

## 4% 法则：退休规划最简单的工具

### 法则由来
威廉·本根 1994 年的论文发现：
- 退休时持有 **股债 50/50** 的组合
- 第一年取出资产的 **4%** 作为生活费
- 此后每年的取出金额随 **3% 通胀** 递增
- 这样的取法，**在历史上至少能撑 30 年而不花光**。

理财规划师迈克尔·基茨对 1870 年以来的数据做回测：**30 年后，4% 法则让资产增长 5 倍的概率大于本金耗尽的概率。**

> 本根同时提示：5% 是上限不合理——某些历史区间下只能撑 20 年就会花光。

### 反推："25 倍法则"——你需要存多少

从 `4% × 总储蓄 = 年度支出` 推导：

```
总储蓄 = 25 × 年度支出
```

这就是第 4 章反复出现的"25 倍年支出"退休目标的来源。

### 考虑社保/养老金后的升级公式

```
总储蓄 = 25 × 年度超额支出
       = 25 × (年度总支出 − 未来稳定收入)
```

**案例**：月花 $4,000、月领社保 $2,000，就只需为另外 $2,000/月（$24,000/年）准备储蓄：
$24,000 × 25 = **$60 万即可退休**。

### 4% 法则仍然适用吗？

- 批评者认为当下利率低，4% 不再安全。
- 但本根本人 2020 年在播客中认为：**低通胀环境下，取现率可能反而能从 4% 提高到 5%**，通胀下降抵消了收益率下降。

### 4% 法则其实偏保守——退休后支出会下降

摩根大通分析 60 万+美国家庭：
- 支出峰值在 **45–49 岁**，之后逐级下降
- 中产家庭（100–200 万美元投资资产）：65–69 岁年均支出 $83,919；75–79 岁 $71,144（下降 15%）
- 消费者支出调查：65–74 岁 $44,897；75 岁以上 $33,740（下降 25%）
- 减少主要集中在：服装、汽车、服务、房贷（因房贷多已还清）

退休研究中心研究：退休后支出每年大约下降 **1%**，而非 4% 法则假设的每年 **+3%**。
从这个角度看，4% 法则对普通退休人员是**额外保守的安全垫**。

## 交叉点法则：更高阶的财务独立定义

来自维姬·罗宾《要钱还是要生活》。
**交叉点** = 月投资收益 ≥ 月支出的那一点。此后你再也不必动用本金。

```
月投资收益 = 交叉资产 × 月收益率
→ 交叉资产 = 月支出 / 月收益率
```

月收益率精确换算：
```
月收益率 = (1 + 年收益率)^(1/12) − 1
```
快速估算可用 `年收益率 / 12`。

### 案例对比

月支出 $4,000（年 $48,000）：

| 预期年收益率 | 交叉点所需资产 | 4% 法则所需资产 |
|:-:|:-:|:-:|
| 3% | **$160 万** | $120 万 |
| 4% | $120 万 | $120 万 |

交叉点法则更保守（不动本金），适合不愿让资产逐年减少、或计划退休 30 年以上的人。

## 更重要的问题：退休后你要做什么

> "对大多数退休人员来说，**身体健康、心理健康和稳固的社会支持**比良好的财务状况更重要。"
> —— 厄尼·泽林斯基《如何自由而快乐地退休》

### 凯文·奥利里（《创智赢家》"神奇先生"）36 岁卖掉第一家公司后
> "我退休三年了，无聊得要命。工作不仅仅是为了钱……工作决定了你是谁，为你提供社交场所……我什么时候退休？我永远不会退休。"

### 作家朱利安·夏皮罗观察卖掉创业公司赚了几百万美元的朋友
> "一年后，他们又回去做原来的项目。他们用钱买了漂亮房子，吃得好。仅此而已。"

### FIRE 运动的一个暗面
作者分享了追随者 "特伦斯" 的自述——两年来靠爱彼迎环游世界：
> "选择成为 FIRE 的一员意味着接受你不再相关或重要，你游离在存在和不存在之间。"

钱可以解决很多问题，但**它无法替代身份、意义和联结**。

## 退休前必须先回答的三个问题

1. **你将如何消磨时间？**
2. **你会与哪些社会群体互动？**
3. **你的最终目标是什么？**

只有这三个问题有了真实答案，"退休"才值得启动——否则账户里的钱不会换来想象中的幸福。

## 本章要点

- **4% 法则**：退休需存 **25 × 年度（超额）支出**；历史数据足以覆盖 30 年。
- **交叉点法则**：更保守、不动本金的财务独立定义：`交叉资产 = 月支出 / 月收益率`。
- 退休后支出实际每年下降 ~1%，说明 4% 法则已相当保守。
- 真正让退休"成功"的不是数字，而是**生活方式**：时间、人、目标。
- 财务独立只解决了问题的一半；知道自己想要什么才是更难的那一半。
